- **Login session preview server restart sonra düşüyor** — `preview_stop` + `preview_start` döngüsünde browser cookie kayboluyor. Kullanıcı her test'te elle login yapmalı. (Çözüm: persistent session kullanan farklı flow, veya preview_start state preservation — şimdilik manuel)
- **Preview lock pattern devam ediyor** — `dotnet build` öncesi `preview_stop` zorunlu. Bu oturumda 3 kez patladı. `dotnet build` hook'u TODO (M-12).
- **Hero'da 👋 emoji** ui-ux-pro-max anti-pattern: "Emoji as icons" Avoid kategorisinde. Sonraki audit'te SVG icon ile değiştirilmeli (ya da text-only "İyi günler, Fikri").
- **`.btn` 34px height** — ui-ux-pro-max touch target min 44×44pt (Apple) / 48×48dp (Material). Mobil için kısa. Plan 03 Faz F sonu mobile responsive review'da düzeltme.
- **Razor `<text>` tag SVG attribute conflict** — `<text x='240' text-anchor='middle'>` Razor block parsing'i kırıyor (RZ1023). `@Html.Raw("<text ...>...</text>")` ile sarılı olmalı.
- **`Views/Logs` git ignored** — `.gitignore`'daki `**/logs/` pattern. Force-add ile (`-f`) commit edildi ama her dokunuşta tekrar uyarı. Pattern düzeltme TODO.
- **`Dashboard/Index.cshtml` 329 satır** — 300 soft eşiği aşıldı. Razor için kural yazılmamış ama prensip aynı (csharp 300/500, JS 250/350). Faz F refactor.
