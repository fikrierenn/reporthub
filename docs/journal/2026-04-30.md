## Dikkat Edilmesi Gerekenler

- **Razor view runtime cache** — view dosyasına dokunulduğunda fresh compile başlar; build OK olsa bile runtime hatalar açığa çıkar (TagHelper RZ1030 modülarize commit'inden öğrenildi). Her view edit sonrası `/Admin/EditReportV2/13` + `/Admin/CreateReportV2` reload smoke zorunlu.
- **Razor `<form>` TagHelper kısıtlaması** — `<form>` elementinde `@@event` (Razor escape) kabul edilmez (RZ1030). Long-form `x-on:event` kullan. Diğer element'lerde (`<div>`, `<button>`) `@@click` çalışır.
- **iframe srcdoc + sandbox `allow-scripts` only** — Chrome null-origin frame'de inline `onclick` engelleniyor. `allow-same-origin` ekleme XSS izolasyonunu zayıflatır ama admin scope'unda kabul.
- **Gridstack `removeAll(true)`** — false ile DOM'da widget birikiyor, true zorunlu. Tab switch'lerde grid rebuild bu pattern'i kullanır.
- **`builder-v2.css` legacy sınıflar** — Plan 04 sonrası `.palette`, `.col-grid-toggle`, `.contract-row` vs. HTML'de yok ama CSS'te duruyor; cleanup commit'i ileri faza.
- **CSS `.col-grid-toggle` HTML'den silindi** ama benim eklediğim CSS henüz orada — render'a etkisi yok ama temizlik gerek.
- **Genişlik (span) UI eksik** — Plan 04'te Setup eski "İleri ayar"daki Genişlik bölümü kaldırıldı, Stil tab'a eklenmedi. Şu an widget genişliği `addWidget`'taki default değerde kalır (KPI: 3, chart: 8, table: 4).
- **Drawer mixin'de `selectedRsColumns`** — selected widget'ın bağlı RS context'inde kolon listesi döner; eğer selected.result null ise boş array. Setup tab Kolon select'i x-show ile selected.result'a bağlı, doğru.
- **`setBoundResult` selected.column ve selected.agg sıfırlar** — RS değiştiğinde yanlış kolon bağı kalmasın diye. `setBoundColumn` agg auto-detect (autoAggregation) yapar + başlık default kolon adına çeker (eğer "Yeni..." prefix'liyse).
- **CreateReportV2 önce klasik akış** — V2 stand-alone tamamlandı ama kullanıcı yine de yeni rapor için klasik `/Admin/CreateReport`'a düşebilir; "Klasik formda düzenle" linki Drawer Rapor Ayarları'nda kalır (rol/kategori/açıklama klasik formda yönetilir, Plan 05 sonrası adım olabilir).
